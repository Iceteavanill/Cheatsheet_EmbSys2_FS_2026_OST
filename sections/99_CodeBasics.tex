\subsubsection{Name mangling}

Name mangling roughly means that the function name is altered in the compiling step.
That is done to resolve namespace issues if 2 functions have the same name but different namespaces.
Used for function overloading.
Because this is only applied in \CC{}, the compiler must adjust the naming scheme if a C library should be called.

\subsection{memcpy}
memcpy copies a fixed number of bytes from one memory location to another.
\begin{lstlisting}
memcpy(destination, source, number_of_bytes);
int a = 42;
int b;
memcpy(&b, &a, sizeof(int)); // b now holds 42
\end{lstlisting}
